# Main Functions Reference

These are the primary entry points for interacting with models in the AI SDK Core.

| Function | Description |
| :--- | :--- |
| `generateText()` | Generate text and call tools from a language model. |
| `streamText()` | Stream text and call tools (useful for chat interfaces). |
| `generateObject()` | Generate structured data (JSON) based on a schema. |
| `streamObject()` | Stream structured data as it is being generated. |
| `embed()` | Generate an embedding for a single value. |
| `embedMany()` | Batch generate embeddings for several values. |
| `generateImage()` | Create images based on text prompts. |
| `experimental_transcribe()` | Convert audio files to text transcripts. |
| `experimental_generateSpeech()` | Convert text into audio speech. |

> See individual documentation links for detailed parameter schemas and return types.
